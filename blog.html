Important FAQ



What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

How do you select an element from the DOM?

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is an event-handling pattern that allows you to handle events at a higher level in the DOM tree other than the level where the event was first received.It is useful because it allows to write cleaner code ans create fewer event listener with similar logic.
